A Quick MySQL How-To

Based on Professor Du's cheatsheet


  1. Connect to mysql server (choose one):
    • $ mysql -u user_id -p
    • $ mysql -u user_id -p database_name
  2. In general to execute a SQL statement online: 
    • mysql> sql_statement;
  3. Create a new database: 
    • mysql> CREATE DATABASE database_name;
  4. Show available databases:
    •  mysql> SHOW DATABASES;
  5. Choose a database:
    •  mysql> USE database_name;
  6. Show table names of the current database: 
    • mysql> SHOW TABLES;
  7. Show schema definition of a table: 
    • mysql> DESCRIBE table_name;
  8. Show the stored procedures names of the current database: 
    • mysql> SHOW PROCEDURE STATUS;
  9. List the contents of a stored procedure: 
    • mysql> SHOW CREATE PROCEDURE STATUS proc_name;
  10. Show all rows of a table: 
    • mysql> SELECT * FROM table_name;
  11. Delete a table:
    • mysql> DROP TABLE table_name;
  12. Execute a SQL script file from the current working directory: 
    • mysql> SOURCE sql_script_file_name;
  13. Load a data file to a table.
    • mysql> LOAD DATA LOCAL INFILE absolute_path_to_the_data_file INTO TABLE table_name;
  14. Exit from mysql:
    • mysql> quit
